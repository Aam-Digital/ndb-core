Configure Note Details Form

Configure the fields for each section of the Note details form. You can edit field properties, add new fields, and drag & drop to reorder. Each section represents a different area of the form.

Top Form Section

Fields displayed at the top of the note form (typically metadata like date, category, etc.)

Middle Form Section

Fields displayed in the main content area (typically subject and text)

Bottom Form Section

Fields displayed at the bottom (typically related entities like children and schools)
